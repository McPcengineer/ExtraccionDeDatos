# ExtraccionDeDatos

Descripción del proyecto

Este proyecto forma parte del Desafío Alura Store – Data Science Latam, cuyo objetivo es realizar un análisis de datos para apoyar la toma de decisiones estratégicas del Sr. Juan, propietario de cuatro tiendas. A través del análisis de ventas y rendimiento, se busca identificar cuál de las tiendas presenta el menor desempeño y debería considerarse para su venta con el fin de invertir en un nuevo negocio.

El análisis se desarrolla utilizando Python y la librería Pandas, y se ejecuta en Google Colab, simulando un escenario real de trabajo para una persona analista de datos.
